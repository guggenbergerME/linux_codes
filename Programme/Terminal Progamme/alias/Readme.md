# Linux alias

## Syntax

Die allgemeine Syntax lautet [1]:

    alias Kurzname='Befehl' 

Beispiel:

    alias ll='ls -la' 

Anzeigen aller definierten Aliase

    alias 

```
alias ll='ls -l'
alias ls='ls --color=auto'
```

Anzeigen eines bestimmten Alias

    alias ll 
    
```
alias ll='ls -l'
```
