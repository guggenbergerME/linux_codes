# Rclone

## install
```
sudo apt-get install rclone-browser
```

## erstes config
```
rclone config
```

## Remote ordner ausgeben
```
rclone lsd [REMOTE]:
```

## Ordner auf dem FTP anlegen
```
rclone mkdir ftp:Verzeichnis/Verzeichnis
```
