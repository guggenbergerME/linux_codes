# Mailcow 

## Installation

### Hostname ändern
Änderung des Host Namens in der mailcow.conf

Docker stoppen
```
docker-compose down
```

Conf ändern
```
/opt/mailcow-dockerized/mailcow.conf
```
Docker starten
```
docker-compose up -d
```

### Verzeichnis
```
# cd /opt/mailcow-dockerized
```
