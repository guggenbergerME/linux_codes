# Wine installieren

## Wine installieren
```
sudo apt-get install wine-stable 
```

## Programme starten
```
wine start /unix /ABSOLUER/PFAD/ZUR/DATEI.exe
```

## Windows-Programme deinstallieren
```
wine uninstaller
```
## Konfiguration
```

```
